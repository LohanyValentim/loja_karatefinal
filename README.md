# Loja de Karatê

Projeto final desenvolvido com front-end, back-end e banco de dados.

## Como executar

1. Acesse a pasta backend:
   cd backend

2. Instale as dependências:
   npm install

3. Inicie o servidor:
   node server.js

4. Abra o arquivo index.html no navegador

## Observação

O sistema exibe produtos cadastrados no banco de dados.
